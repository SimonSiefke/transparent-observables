Frameworks vs Libraries

The difference between frameworks and libraries is good described by this quote:

`You use the library, but the framework uses you`

The framework dictates the way the code looks and enforces specific design patterns while exempting others.

```js
import _ from 'lodash'

_.intersection([2, 1], [2, 3])
// => [2]
```
